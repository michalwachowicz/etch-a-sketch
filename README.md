# Etch-a-Sketch

> A simple grid-based Etch-a-Sketch toy inside your browser

## Built with

- HTML
- SASS
- Javascript

## Usage

- Hover over grid squares with your mouse to paint them
- Reset canvas with the button below it and specify the new size of grid
